== Description ==

SkyLearn Flashcards is a powerful WordPress plugin that allows you to create engaging, interactive flashcard sets for educational purposes. Whether you're a teacher, student, or running an online academy, this plugin provides everything you need to create effective learning experiences.

**Key Features:**

* **Easy Flashcard Creation** - Create unlimited flashcard sets with an intuitive editor
* **Interactive Learning** - Smooth flip animations and engaging user interface
* **Multiple Study Modes** - Normal, quiz, review, and random study modes
* **Progress Tracking** - Track learning progress and performance
* **Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
* **LMS Integration** - Compatible with LearnDash, TutorLMS, and other popular LMS plugins
* **Shortcode Support** - Easy integration into posts, pages, and courses
* **Customizable Styling** - Match your site's branding with customizable colors and themes

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/skylearn-flashcards` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the SkyLearn Flashcards menu in your WordPress admin to create your first flashcard set.
4. Use the `[skylearn_flashcards id="X"]` shortcode to display your flashcards on any post or page.

== Frequently Asked Questions ==

= How do I create my first flashcard set? =

After activating the plugin, go to SkyLearn Flashcards in your WordPress admin menu. Click "Add New Set" and start creating your flashcards by adding questions and answers.

= Can I use this with my LMS? =

Yes! SkyLearn Flashcards integrates seamlessly with popular LMS plugins like LearnDash and TutorLMS. Your flashcards can be embedded directly into courses and lessons.

= Is there a limit to how many flashcards I can create? =

The free version allows you to create multiple flashcard sets. The premium version removes all limits and adds advanced features like detailed reporting and bulk export.

= Can I customize the appearance? =

Absolutely! The plugin includes customizable color schemes and styling options. You can match the flashcards to your site's branding.

= Does it work on mobile devices? =

Yes, the plugin is fully responsive and works great on all devices including smartphones and tablets.

= How do I upgrade to premium? =

Visit our website at https://skyian.com/skylearn-flashcards/ to purchase a premium license and unlock all advanced features.
