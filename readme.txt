== Description ==

SkyLearn Flashcards is a comprehensive flashcard plugin designed for educators, students, and online learning platforms. Create engaging, interactive flashcard experiences with powerful features and seamless LMS integration.

**Key Features:**

* **Interactive Flashcard Sets** - Create unlimited flashcard sets with rich content support
* **Responsive Design** - Mobile-friendly interface that works on all devices  
* **LMS Integration** - Seamless integration with LearnDash, TutorLMS, and other popular LMS platforms
* **Lead Collection** - Built-in forms to capture student information and grow your audience
* **Progress Tracking** - Track student performance and learning progress
* **Customizable Styling** - Match your site's branding with flexible color schemes
* **Export & Import** - Bulk export/import functionality for easy content management
* **Analytics Dashboard** - Detailed reporting on flashcard usage and student performance

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/skylearn-flashcards` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the SkyLearn Flashcards menu item to configure the plugin
4. Create your first flashcard set and embed it using shortcodes or blocks

== Frequently Asked Questions ==

= Is this plugin free? =

The core plugin is free with essential flashcard functionality. Premium features like advanced reporting, bulk export, and lead collection require a premium license.

= Does it work with my LMS? =

Yes! SkyLearn Flashcards integrates with popular LMS platforms including LearnDash and TutorLMS. More integrations are coming soon.

= Can I import existing flashcards? =

Yes, the premium version includes bulk import/export functionality to help you migrate from other platforms or backup your content.

= Is it mobile-friendly? =

Absolutely! The plugin is fully responsive and provides an excellent experience on all devices.

= Do you provide support? =

Yes, we provide support for all users. Premium users receive priority support with faster response times.
